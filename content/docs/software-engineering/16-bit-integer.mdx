---
title: 16-bit Integer
description: Integers
---

A 16-bit integer can hold:

### Signed 16-bit integer

- Minimum: -32,768 (-2^15)
- Maximum: 32,767 (2^15 - 1)

### Unsigned 16-bit integer

- Minimum: 0
- Maximum: 65,535 (2^16 - 1)

Signed uses one bit for the sign, leaving 15 bits for magnitude. Unsigned uses all 16 bits for positive values.
